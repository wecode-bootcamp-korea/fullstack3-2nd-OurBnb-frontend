Convention
------
Branch
- Feature/<기능 or 페이지> : 새로 만들 때 사용
- Fix/<고치는 것> : 수정, 개선할 때 사용

Commit
- Add : 기능 추가할 때
- Fix : 기능 수정, 개선할 때
- Del : 기능 삭제할 때
- Merge : 파일 머지할 때

File
- 페이지 폴더 및 파일명 : Pascal Case
- 컴포넌트 파일명 : Pascal Case
- 클래스명 : camel Case
